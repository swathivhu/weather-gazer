# **App Name**: WeatherGazer

## Core Features:

- Current Weather Display: Fetches and displays current weather data (city, country, temperature, condition, icon, humidity, wind speed, 'feels like' temperature) from WeatherAPI.com based on user-entered location.
- 3-Day Forecast Display: Fetches and displays a 3-day weather forecast (date, max temperature, min temperature, condition, icon) from WeatherAPI.com based on user-entered location.
- Location Input and Search: Allows users to input a city or latitudinal/longitudinal coordinates to search for weather data.
- Tab Navigation: Enables users to switch between 'Current Weather' and '3-Day Forecast' views via tab navigation.
- Error Handling: Implements error handling for invalid location inputs or API request failures.

## Style Guidelines:

- Primary color: Light purple (#A096D3), suggesting a calming and somewhat mysterious mood, suitable for the topic of weather.
- Background color: Very light purple (#F3F2F7), near white.
- Accent color: Violet (#8575C3) for highlights and interactive elements.
- Font: 'Inter', a grotesque-style sans-serif font, for both headlines and body text.
- Glassmorphic main card with backdrop-filter (blur of 15px), a translucent white overlay (rgba(255,255,255,0.15)), border-radius of 25px, and a subtle shadow for a modern, frosted-glass effect. The layout will be fully responsive.
- Subtle transitions and hover effects for a smooth user experience; tab switching will have a smooth animation. Full screen background uses a gentle animated gradient.
- Weather icons should be simple, modern, and consistent in style, matching the overall aesthetic of the app.
