# 🌦 WeatherGazer

A modern, responsive weather dashboard built using **Next.js, TypeScript, and Tailwind CSS**.

WeatherGazer provides real-time weather data, 3-day forecasts, hourly insights, and dynamic weather tips in a clean glassmorphism UI.

---

## 🚀 Features

- 🌍 Search weather by city
- 🌡 Real-time temperature display
- 📅 3-day weather forecast
- 📊 Temperature visualization chart
- 💡 Dynamic weather insights
- 🎨 Modern glassmorphism UI
- ⚡ Fast and responsive design

---

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Weather API Integration
- Firebase Studio
